# Tooling Guide

## Serena
Use Serena for repository inspection and code navigation.

Valid known commands:
- serena --help
- serena project --help
- serena project index D:\utility-disbursement-app

Do not assume invalid commands such as:
- serena activate
- serena status

## CodeGraph
Use CodeGraph only for codebase visualization / dependency inspection.

Do not let CodeGraph define architecture.
Use it as inspection support only.

## Skills
Skills are stored under:

.ai/skills/

Use them as workflow guidance only.
Do not import old ai_runtime governance or execution-contract system into this app.
