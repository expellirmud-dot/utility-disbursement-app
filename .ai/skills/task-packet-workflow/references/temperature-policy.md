# Temperature Policy

This repository uses low-randomness AI execution.

Recommended settings:

- Planning: 0.1 to 0.3
- Coding: 0.1 to 0.2
- QA: 0.0 to 0.1

If a tool does not expose temperature configuration, the agent must behave conservatively:

- no speculative implementation
- no creative architecture
- no extra feature work
- no unassigned file edits
